# Torch Frontend

This is the frontend application for **The Torch**, built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com/), and [Sanity](https://www.sanity.io/).

## Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS v4, Radix UI primitives
- **CMS:** Sanity
- **Database / Auth:** Supabase
- **State Management:** Jotai
- **Email:** Resend
- **Messaging/Queues:** Upstash QStash

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

Make sure to set up your `.env.local` file. You can use `.env.example` as a reference for required variables.
